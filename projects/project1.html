Project Overview

Developed an AI-enhanced University Reporting System for my Final Year Project, enabling students to easily report campus issues (Dorm, Faculty, Campus Grounds) and allowing administrators and staff to manage resolutions effectively.

Technologies involved: AI/ML, Google Maps API, Flutter, JavaScript, Firebase. This project honed my skills in full-stack development, AI integration, and building multi-user workflow systems.

Key Features

  • Feature 1: AI-driven categorization of reported problems based on user descriptions (model hosted on Render).
  • Feature 2: Google Maps API integration for precise location reporting on campus.
  • Feature 3: Image uploads for visual context.
  • Feature 4: Report tracking (Pending, In Progress, Resolved) for students.
  • Feature 5: Ticket submission system for unresolved issues.
  • Feature 6: Admin dashboard for report assignment and status management.
  • Feature 6: Staff portal to view assigned tasks and submit proof of resolution.
