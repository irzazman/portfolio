Project Overview

Led the development of an AI-powered real-time sign language mobile application for my Workshop 2 project at UTeM, aimed at assisting deaf travelers.

Key Features

  • Feature 1: Real-time sign language translation using an AI model.
  • Feature 2: An educational module for learning sign language.
  • Feature 3: A gamified learning section.
  • Feature 4: A support system connected to a web admin dashboard.
